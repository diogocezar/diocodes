## 📦 Instalação

Para instalar basta utilizar o pnpm para instalar as dependências do projeto.

```bash
pnpm install
```

## 🚀 Rodando o projeto localmente

Para rodar o projeto localmente basta utilizar o comando `pnpm dev`.

```bash
pnpm dev
```
